AWSの検証用に以下のようなシナリオを考えています。
現実的なシナリオといえますか？
実装言語は.NET 8.0を想定しています。

１．AWSCognitoで認証してトークンを取得
２．トークンを使用してAppSyncをコール
３．AppSyncからLambda関数をコールする
４．Lambda関数からDynamoDBにユーザー情報とAppSyncの入力値を保存する

Client
  ↓ Cognito Hosted UI / OAuth2 / SRP などでログイン
Cognito User Pool
  ↓ ID Token / Access Token
Client
  ↓ Authorization: Bearer <token>
AppSync GraphQL API
  ↓ Resolver
Lambda (.NET 8)
  ↓ PutItem / UpdateItem
DynamoDB
